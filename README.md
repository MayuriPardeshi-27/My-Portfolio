# My Portfolio Website

A personal portfolio website to showcase my projects, skills, and contact information. Designed to highlight my front-end development abilities and provide a professional online presence.

## Features
- Home page with introduction
- About section with skills and education
- Projects showcase 
- Contact form with EmailJS integration
- Responsive design for desktop and mobile
- Smooth scrolling and animations for better user experience

## How to Use
1. Open `index.html` in your browser
2. Navigate through different sections: Home, About, Projects, Contact
3. Use the contact form to send messages (EmailJS required)
4. Explore projects 

## Technologies Used
- HTML
- CSS
- JavaScript
- EmailJS (for contact form)
